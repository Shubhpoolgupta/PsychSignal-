# PsychSignal
 # 🧠 PsychSignal - Mental Health Text Classification System

## 📌 Overview

PsychSignal is an AI-powered application that analyzes user-provided text and predicts the underlying mental health condition using Natural Language Processing (NLP), Machine Learning, and Deep Learning techniques.

The system is capable of identifying multiple mental health states from textual input and provides real-time predictions through an interactive Streamlit web application.

## 🎯 Problem Statement

Mental health issues often manifest through written communication, social media posts, journals, and conversations. Early detection of potential mental health concerns can assist individuals in seeking appropriate support and resources.

This project aims to classify text into different mental health categories by leveraging state-of-the-art NLP techniques and transformer-based models.

---

## 🚀 Features

* Text-based mental health assessment
* Real-time predictions through Streamlit UI
* NLP-based text preprocessing pipeline

* Performance evaluation using Classification Report and Confusion Matrix

## 🧠 Mental Health Categories

The model predicts one of the following mental health states:

1. Normal
2. Depression
3. Suicidal
4. Anxiety
5. Bipolar
6. Stress
7. Personality disorder

## 🤖 Model Development

### Machine Learning Approach

Traditional ML models were trained and evaluated on the processed dataset.

### Deep Learning Approach

A pre-trained BERT model (`bert-base-uncased`) from Hugging Face was fine-tuned for mental health text classification.

Key techniques used:

* Transfer Learning
* Fine-Tuning
* Tokenization using BERT Tokenizer
* Classification Head Customization
* Model Evaluation
